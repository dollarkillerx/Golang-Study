死循环
- SDK Math包
    - 四舍五入
        ``` 
        // 四舍五入
        fmt.Println(math.Round(3.4))
        fmt.Println(math.Round(3.5))
        ```
    - Abs绝对值
        ``` 
        fmt.Println(math.Abs(-335))
        ```
    - 开方&&乘方
        ``` 
        // 乘方
        fmt.Println(math.Pow(2,3))
        // 开方
        fmt.Println(math.Sqrt(9))
        // 开平方
        fmt.Println(math.Pow(8,(1.0/3)))
        ```
    - 取整
        ``` 
        向下取整
        math.Floor()
        向上取整
        math.Ceil(3.01)
        ```
